# svg-captcha-nodejs
Genarate SVG captcha use nodejs
----
### Install and start:
```
run: npm install
run: npm start
```
----
#### Go to http://localhost:5000/captchapreview?size=6&color=true&charPreset=1234567890
----
#### demo: https://svg-captcha.herokuapp.com/
